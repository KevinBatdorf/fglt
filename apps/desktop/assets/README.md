# Desktop assets

Generated assets for the Windows desktop build.

| File | Purpose | Source |
|---|---|---|
| `icon-1024.png` | Master 1024×1024 raster (also doubles as README hero) | Generated from `logo.png` at the repo root |
| `icon.ico` | Windows app icon — multi-size 16/32/48/64/128/256 | Generated from `icon-1024.png` |
| `docker-compose.consumer.yml` | Bundled into the binary so the app can manage the Docker stack itself | Copied from the repo root by `bun run build:assets` |
| `backend/` | Backend source bundled into the binary so the consumer's compose `build: ./backend` resolves without a registry | Generated by `bun run build:backend` |

Re-run `bun run build:icons` whenever `logo.png` changes.

```bash
cd apps/desktop
bun run build:icons
```
